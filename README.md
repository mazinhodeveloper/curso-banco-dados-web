## Banco de Dados para Web - 84 horas  
- Modelar a estruturar o banco de dados, seguindo os requisitos de uma aplicação web
- Preparar o ambiente de desenvolvimento, conforme requisitos do Sistema de Gerenciamento de Banco de Dados (SGBD)
- Estruturar banco de dados físicos, utilizando linguagem de manipulação de dados
- Integrar banco de dados à aplicação web de acordo com os requisitos do projeto
- Manipular dados a partir da aplicação, utilizando linguagem de programação back-end e integração ao banco de dados, com base na necessidade do projeto    

### Objetivo 
Aprender a aplicar e manipular dados em um sistema gerenciador de banco de dados, conectando com a linguagem de programação.    
Este curso faz parte do Técnico em Informática para Internet. Por isso, se quiser se inscrever neste curso técnico no futuro, você pode ter o benefício de não precisar cumprir essa carga horária novamente.    

### Programa 
Modelar, estruturar, preparar, integrar, manipular dados a partir da aplicação, utilizando linguagem de programação back-end e integração ao banco de dados.    

### Público 
Este curso é para pessoas que se identificam com a área de tecnologia da informação e tem interesse em saber mais sobre linguagens de programação e como realizar suporte a banco de dados para aplicações web e mobile.
